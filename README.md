# Vision-Based Augmented Reality Assistant for Optimized Furniture Layout

## Overview

This project is a mobile Augmented Reality (AR) application developed using Unity and AR Foundation. It enables users to visualize virtual furniture in their real environment before purchasing, helping them make better furniture placement decisions and reducing incorrect purchases.

---

## Features

- Markerless AR Plane Detection
- Real-Time Furniture Placement
- Move Furniture
- Rotate Furniture
- Scale Furniture
- Delete Furniture
- Multiple Furniture Selection
- Android Gallery Image Picker
- Interactive Touch Gestures

---

## Technologies Used

- Unity 2022 LTS
- C#
- AR Foundation
- ARCore XR Plugin
- Unity XR Interaction
- Native Gallery
- Git
- GitHub

---

## Project Workflow

1. Scan the floor using the mobile camera.
2. Detect horizontal surfaces.
3. Select furniture from the menu.
4. Place furniture on the detected plane.
5. Move, rotate, or resize furniture.
6. Delete furniture if required.
